1. GET

GET

A GET request is an HTTP method used to retrieve data from a server. When you enter a URL in your web browser, it sends a GET request to the server to fetch the web page. GET requests are read-only and do not alter any data on the server. They can include query parameters to filter or specify the data needed. For example, 'GET /users?name=John' fetches users with the name John. GET requests are also idempotent, meaning they can be repeated without side effects.
