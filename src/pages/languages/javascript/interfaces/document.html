1. Document

Document

The Document interface in JavaScript represents the entire HTML or XML document that is loaded into a web browser. It serves as the primary entry point for interacting with the web page's content and structure through the Document Object Model (DOM). The document object provides numerous properties and methods for accessing and manipulating elements within the document.

Syntax

document.methodName(identifier);
document.methodName(identifier).property = value;
document.methodName(identifier).property.attribute = value;
document.methodName.attribute(identifier);
document.methodName(identifier).attribute(identifier, value);
document.methodName(identifier).attribute(identifier);

These are the general syntax you can expect using the document interface. You can see that it is always followed by the methodName but can then vary depending on it's usage.

Examples

<body>
    <div id="myDiv">
        <p>This is text within the div</p>
    </div>

    <script>
        var newText = document.createElement('p');
        newText.innerHTML - 'Adding additional text';
    </script>
</body>

Let's start going through examples using this as our starting HTML/JavaScript body code.

document.getElementById('myDiv');

When this is executed, the method searches the DOM for an element with the id attribute equal to 'myDiv'. If such an element is found, the method returns a reference to that element. If no element with that id exists, it returns null.

document.getElementById('myDiv').innerHTML ='<p>Changing the HTML within the myDiv element</p>';

This method accesses the element with id="myDiv" and sets its inner HTML content to the provided string, effectively updating the content displayed on the web page within that element.

document.getElementById('myDiv').style.backgroundColor = '#FFFFFF';

This method accesses the div element with id="myDiv" and sets its background color to white using inline CSS styles.

document.getElementById('myDiv').appendChild(newText);

This method accesses the div element with id="myDiv" and appends the newText element as a child.

document.getElementById('myDiv').setAttribute('data-info', 'This is a div element');

This method accesses the div element with id="myDiv" and sets a custom attribute data-info with the value 'This is a div element'.

document.getElementById('myDiv').getAttribute('data-info');

This method accesses the div element with id="myDiv" and retrieves the value of the data-info attribute.

Categories

The Document interface atrributes and usages can be broken down into 3 categories.

Instance Properties

Instance properties of the Document interface are attributes or variables that hold data or state about the document. These properties provide information about the document and allow for interaction with its structure and content.

Instance Methods

Instance methods of the Document interface are functions that perform actions on the document or its elements. These methods allow you to manipulate the documentâ€™s structure, content, and style dynamically.

var element = document.getElementById('myDiv'); // Get the element with id="myDiv"
var newDiv = document.createElement('div'); // Create a new div element
var element = document.querySelector('.myClass'); // Get the first element with class="myClass"

Events

Events are actions or occurrences that happen in the document, such as user interactions, browser actions, or other events like loading or resizing. The Document interface can listen for these events and execute code in response to them.
