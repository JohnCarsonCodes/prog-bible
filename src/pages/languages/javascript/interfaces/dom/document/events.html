1. Events

Events

Events in the Document interface are actions or occurrences that happen in the web page or browser, such as user interactions, browser actions, or other occurrences like loading or resizing. Events are a core part of the web's interaction model, allowing developers to create dynamic, interactive experiences by responding to these occurrences. JavaScript provides a robust event handling system through the Document Object Model (DOM), enabling developers to capture and respond to events effectively.

Examples

// Execute code when the DOM is fully loaded
document.addEventListener('DOMContentLoaded', function() {
    console.log('The DOM is fully loaded.');
});

// Execute code when the myDiv element is clicked
document.getElementById('myDiv').addEventListener('click', function() {
    console.log('myDiv was clicked.');
}); 

// Execute code when a key is pressed
document.addEventListener('keydown', function(event) {
    console.log('A key was pressed: ' + event.key);
});
