1. getElementById()

getElementById()

The getElementById() method is a widely used function provided by the Document interface in JavaScript. It allows you to access a single element in the DOM based on its unique id attribute. This method is highly efficient and straightforward, making it a fundamental tool for DOM manipulation.

vs. querySelector()

Compared to querySelector(), it is faster and more efficient since it directly accesses elements by their unique id. Use getElementById() when needing to Select by Unique ID, Performance-Critial Tasks or for Simplicity and Clarity

document.getElementById('myDiv'); // Unique ID
document.getElementById('submitButton'); // Performance-Critical Tasks
document.getElementById('mainHeader'); // Simplicity and Clarity
Flexibility
  • Limited to selecting elements by their unique id attribute.

Syntax

document.getElementById(id);

An id is a string representing the id attribute of the element you want to select. This id should be unique within the document

Examples

document.getElementById('myDiv'); // Basic Usage
document.getElementById('myDiv').innerHTML = 'Updated information'; // Selecting the div ID and updating the enlclosed content
document.getElementById('myDiv').style.color = 'blue'; // Selecting the div ID and changing the color to blue

// Selecting a button element to add an event listener
var button = document.getElementById('myButton');
button.addEventListener('click', function() {
    alert('Button was clicked!');
});

Use Case

Accessing Form Elements

Selects the <input> element with the id of username and sets its value to 'JohnDoe'.

// HTML
<input type="text" id="username" />

// JavaScript
var usernameInput = document.getElementById('username');
usernameInput.value = 'JohnDoe';

Toggling Visibility

This code selects the element with the id of myDiv, hides it by setting its display style to 'none', and then immediately shows it again by setting its display style to 'block'.

// HTML
<div id="myDiv">Content to hide or show</div>

// JavaScript
var element = document.getElementById('myDiv');
element.style.display = 'none'; // Hides the element
element.style.display = 'block'; // Shows the element

Updating Attributes

Selects the <img> element with the id of myImage and changes its src attribute from 'image1.jpg' to 'image2.jpg'.

// HTML
<img id="myImage" src="image1.jpg" />
    
// JavaScript
var image = document.getElementById('myImage');
image.setAttribute('src', 'image2.jpg');

Tips

Ensure Unique IDs

Make sure that the id you are targeting is unique within the document to avoid unexpected behavior.

// Correct usage
<div id="uniqueId">Content</div>

// Incorrect usage
<div id="duplicateId">First</div>
<div id="duplicateId">Second</div>

Check For NULL

Always check if getElementById() returns null before trying to manipulate the element. This helps avoid runtime errors.

var element = document.getElementById('nonExistentId');
if (element) {
    element.innerHTML = 'This will not run if element is null';
}

Use Descriptive IDs

Use meaningful and descriptive id names to make your code more readable and maintainable.

// Descriptive ID
<div id="userProfile">Profile Content</div>

// Non-descriptive ID
<div id="up5168">Profile Content</div>

Performance

getElementById() is very fast and efficient because it directly accesses elements by their unique id. Use it whenever you need to access a single, specific element quickly.

Naming Consistency

Use consistent naming conventions for id attributes across your document. For example, use camelCase, kebab-case, or snake_case consistently.

// Camel case
<div id="userProfile"></div>

// Kebab case
<div id="user-profile"></div>

// Snake case
<div id="user_profile"></div>
