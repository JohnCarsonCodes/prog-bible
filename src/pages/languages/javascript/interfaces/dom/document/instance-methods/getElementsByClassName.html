1. getElementsByClassName()

getElementsByClassName()

The getElementsByClassName() method is provided by the Document interface in JavaScript. It returns a live HTMLCollection of all elements in the document that have all the specified class names. This method is useful for selecting multiple elements that share a common class name.

vs. querySelectyAll()

Compared to querySelectorAll(), it is slightly faster to select elements by class name due to its simplicity. Use getElementsByClassName() when you prefer a live collection that updates automatically with changes to the DOM.

Flexibility
  • Returns a live HTMLCollection of all elements in the document with the specified class name(s).
  • The HTMLCollection is automatically updated when the document changes.

Syntax

document.getElementsByClassName(classNames);

classNamesis a string containing one or more class names separated by spaces. Only elements that have all the specified class names will be selected.

Examples

<!-- HTML -->
<div class="myClass class1">Item 1</div>
<div class="myClass">Item 2</div>
                        document.getElementsByClassName('myClass'); // An HTMLCollection of elements with class "myClass"
document.getElementsByClassName('class1 class2'); // An HTMLCollection of elements with both "myClass" and "class1"

Use Case

Styling Multiple Elements

Selects all elements with the class name highlight and sets their background color to yellow.

var elements = document.getElementsByClassName('highlight');
for (var i = 0; i < elements.length; i++) {
    elements[i].style.backgroundColor = 'yellow';
}

Form Validation

Selects all elements with the class name required and adds a red border to those elements that have an empty value.

var elements = document.getElementsByClassName('required');
for (var i = 0; i < elements.length; i++) {
    if (elements[i].value === '') {
        elements[i].style.border = '1px solid red';
    }
}

Updating Content Dynamically

Selects all elements with the class name update and changes their text content to "New Content".

var elements = document.getElementsByClassName('update');
for (var i = 0; i < elements.length; i++) {
    elements[i].textContent = 'New Content';
}

Tips

Live Collection

Remember that getElementsByClassName() returns a live HTMLCollection. This means that the collection is automatically updated when the document changes.

Use With Care

Be cautious when modifying the DOM inside loops iterating over the HTMLCollection, as the live nature of the collection can lead to unexpected behavior.

var elements = document.getElementsByClassName('myClass');
for (var i = 0; i < elements.length; i++) {
    // Modifying the DOM here can affect the live collection
}

Convert to Array for Convenience

Convert the live HTMLCollection to an array if you need to use array methods like forEach, map, or filter.

var elementsArray = Array.prototype.slice.call(document.getElementsByClassName('myClass'));
elementsArray.forEach(function(element) {
    element.style.color = 'green';
});

Specificity of Class Names

Use specific class names to avoid selecting unintended elements. Ensure that the class names used are unique enough to target only the desired elements.
