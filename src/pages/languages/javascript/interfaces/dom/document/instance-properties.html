1. Instance Properties

Instance Properties

Instance properties of the Document interface are built-in attributes that hold data or state about the document. These properties provide information about various aspects of the document and allow for interaction with its structure, content, and metadata. Understanding and utilizing these properties is crucial for effective DOM manipulation and dynamic web development.

Examples

console.log(document.title); // Outputs the current document title
document.title = "New Title"; // Sets the document title to "New Title"
console.log(document.body); // Outputs the <body> element
console.log(document.URL); // Outputs the current document URL
