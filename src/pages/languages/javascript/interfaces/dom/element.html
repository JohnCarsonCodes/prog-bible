1. Events

Events

Cheatsheet

Mouse
click Triggered when an element is clicked
wheel Triggered when the mouse wheel is scrolled
mousedown/mouseup Triggered when a mouse button is pressed/released over an element
mouseover/mouseout Triggered when the mouse pointer enters/leaves an element
mousemove Triggered whenever the mouse pointer is moved within an element
dblclick Triggered when an element is double-clicked
contextmenu Triggered when the right mouse button is clicked to open a context menu
Keyboard
keydown Triggered when a key is pressed down
keyup Triggered when a key is released
keypress* Triggered when a key is pressed and held down (deprecated)
